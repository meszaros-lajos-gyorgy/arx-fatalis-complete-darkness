# Arx Fatalis - Complete Darkness

## Installation

Just download this repo as a zip and unpack it into your Arx Fatalis/Arx Libertatis game folder

## Uninstall

Remove `graph` folder from your Arx Fatalis/Arx Libertatis game folder
